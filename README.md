# 감정 일기장 (Emotion diary)
사용자가 날짜, 감정, 일기를 기록할 수 있는 간단한 웹입니다. 입력된 일기 내용은 브라우저 콘솔에 출력됩니다. 

## 기능
- 날짜 입력
- 감정 선택(행복, 평범, 슬픔, 화남, 설렘, 피곤)
- 일기 작성
- 저장 버튼 클릭 시 콘솔에 입력 내용 출력
  
## 사용 기술
HTML / CSS / JavaScript / React
